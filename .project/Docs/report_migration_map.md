## Key Definitions

### Runtime-Generated
- **Criteria:** Auto-generated by build/test systems, performance collectors, or policy gates
- **Timestamps:** Usually contain ISO-8601 or numeric timestamps (e.g., `20260307T113732Z`)
- **Format:** JSON, JSONL, PNG charts
- **Mutability:** Can be safely deleted/regenerated
- **Audience:** System diagnostics and telemetry

### Curated
- **Criteria:** Hand-authored or explicitly maintained by team members
- **Content:** Summary reports, decision records, periodic digests
- **Format:** Markdown, with human narrative
- **Naming:** Follows `Report_*` prefix convention
- **Audience:** Team documentation and historical record
