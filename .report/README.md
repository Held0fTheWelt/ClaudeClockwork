# Curated Reports

This directory contains **human-reviewed, curated reports only**.

## Policy

- ✅ **Curated reports:** markdown summaries, analysis documents, release certificates, approved findings
- ❌ **Machine-generated noise:** transient outputs, raw logs, intermediate artifacts
- ❌ **Absolute host paths:** security redaction required before publication

Machine-generated outputs and runtime artifacts go to `.clockwork_runtime/` instead.

## Contents

- `performance/` — curated performance summaries
- `routing/` — curated routing analysis reports
- Other markdown files — approved project reports

## Related Directories

- **`.clockwork_runtime/`** — runtime outputs, ledgers, eval results, evidence bundles
- **`.project/Docs/`** — project-operational docs (plans, reviews, critics)
