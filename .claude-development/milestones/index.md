1. Create `milestones/M<N>_<name>_<YYYY-MM-DD>.md`
2. Add a row to the table above
3. In the plan reference only MVP IDs from the chain; add new MVP text in **Clockwork_MVP_Chain.md** (see [../MVP_RULE.md](../MVP_RULE.md))
