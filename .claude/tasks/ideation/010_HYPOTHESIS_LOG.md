# Task: Hypothesis Log + Experiment Budget

## Goal
Turn creative ideas into testable hypotheses with a strict experiment budget.

## Rules
- Max 3 hypotheses.
- Max 3 experiments per run (see `governance/experiment_budget.md`).
- Experiments must be small: 1 command or <= 5 short steps.

## Output
- HypothesisSpec
- ExperimentSpec
- Evidence folder: `validation_runs/YYYY-MM-DD/experiments/`
