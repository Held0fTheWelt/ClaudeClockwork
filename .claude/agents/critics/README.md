# Critics (Board)

This board runs multiple specialized critics. Each critic must:
- point out risks / gaps
- propose concrete mitigations
- be explicit about limitations and evidence

Critics:
- systemic
- technical
- legal
- security
- moral
- creative
- methodical

Outputs should be saved as JSON panel files under `.claude-performance/reviews/`.
See `skills/playbooks/critics_board.md`.
