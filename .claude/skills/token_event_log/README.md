# token_event_log

Append a single token spend event (`budget_event`) to a JSONL file under `.claude-performance/events/`.

## Why
Budgeting accuracy depends on good event logging. This skill provides a standard schema and file location.

## Skill ID
- `token_event_log`

## Wrapper
If you prefer a CLI wrapper, use:
- `.claude/tools/token_event_autologger.py`
