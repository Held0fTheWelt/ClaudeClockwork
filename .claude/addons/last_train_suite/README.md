# Last-Train Suite (Pack)

## Included skills
- `last_train_merge` — scan multiple zip archives, produce evolution/loss timeline, and build a combined zip.

## Notes
- Input zips should be ordered oldest -> newest.
- Cache artifacts are ignored by default.

## Boundaries
- This pack is optional. Core should not depend on it to boot.
- See `../BOUNDARIES.md` and `../map.yaml`.
