# Add-ons (Packs)

Diese Ordner sind **kompakte Einstiegsseiten** für "Pakete" aus Skills + Tasks + Templates.
Sie enthalten keine doppelte Implementierung, nur Orientierung.

Packs:
- `docforge_pdf_quality/` — hochwertige Dokumente + PDF Render
- `cleaning_suite/` — Repo-/Code-Cleaning + Apply
- `last_train_suite/` — ZIP Evolution + Merge
- `shadow_prompts/` — kompakte Shadow-Prompt-Sets
- `performance_budgeting/` — token budget analysis + efficiency review
- `critics_suite/` — multi-critic board (legal/security/moral/creative/methodical)
- `model_routing_personaler/` — Personaler model selection + learning hit list
